### 6. 🔱 Lưu trữ (`/my-forks-archive`)
- Các bản fork từ các dự án mã nguồn mở khác.

## 🚩 Quy chuẩn Đẩy mã (Push Standards)

- **Không Push**: `node_modules`, `venv`, `bin`, `obj`, `.env`.
- **Git LFS**: Sử dụng cho các file binary, model AI, và video lớn.
- **README**: Mỗi thư mục chuyên mục đều có tệp README riêng giải thích nội dung.

# 🗺️ Repository Ecosystem Map (Arter2507)

This file provides an overview of the new organization structure for repositories on GitHub.

## 📂 Root Directories & Classification

### 1. 🎓 Education & Learning (`/edu`)
- **`java/`**: Java projects (EJB, JSF, Sem4, Spring).
- **`dotnet/`**: .NET MVC and eProject applications.
- **`html-css/`**: Web templates and interface projects.
- **`python/`**: Machine Learning and Automation research.
- **`mobile/`**: Flutter sample applications.
- **`snippets/`**: Small code snippets and quick reference documentation.

### 2. 🛠️ Utilities & Bots (`/utilities-tools-bots`)
- Contains productivity tools, Chrome Extensions, and Discord Bots.

### 3. 📖 Guides & Documentation (`/guides`)
- Specialized tutorial files (e.g., Justice Online).

### 4. 🌟 Featured Projects (Root or `/utilities-tools-bots`)
- **`create-heyai-ruleset`**: AI ruleset management.
- **`app-person-hub`**: Personal hub ecosystem.

### 5. 🌌 Cosmic Planner (`/cosmic-planner`)
- Merged `frontend` and `hub` source code for the Cosmic Planner app.

### 6. 🔱 Archive (`/my-forks-archive`)
- Forked repositories from other open-source projects.

## 🚩 Push Standards

- **Do Not Push**: `node_modules`, `venv`, `bin`, `obj`, `.env`.
- **Git LFS**: Use for binary files, AI models, and large videos.
- **README**: Each category folder contains its own README explaining its contents.
